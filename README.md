# 🔐 Sistema de Login em PHP

Sistema simples de autenticação desenvolvido em **PHP**, permitindo gerenciar usuários através de um painel com funcionalidades básicas de controle de acesso.

## 📋 Funcionalidades

* 🔑 **Login de usuário**
* ➕ **Adicionar novos usuários**
* ❌ **Excluir usuários**
* ✔️ **Validação de login**
* 🚪 **Logout (sair da conta)**

O sistema permite gerenciar acessos de forma simples, sendo ideal para aprendizado de **autenticação e controle de usuários em PHP**.

## 🛠️ Tecnologias Utilizadas

* PHP
* HTML
* CSS
* MySQL 
* XAMPP / WAMP para ambiente local

## 📂 Estrutura do Projeto

* `login.php` → Tela de login
* `valida_login.php` → Validação das credenciais
* `usuario_add.php` → Cadastro de novos usuários
* `usuario_delete.php` → Remoção de usuários
* `logout.php` → Encerrar sessão do usuário

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para praticar conceitos de:

* Autenticação de usuários
* Manipulação de sessões em PHP
* Estruturação de sistemas web simples
